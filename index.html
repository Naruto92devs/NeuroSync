    VPS 

    A virtual private server (VPS) is a machine that contains all the software and all the data necessary to run the application's website. It is called virtual because it consumes only part of the physical resources of the server managed by a third-party provider.

    GPU

    Most often, a cloud GPU server is a cloud-based virtualization service with a graphics accelerator.
    As in other cases, the GPU allows the cloud to run faster, provides high reliability and fault tolerance of the environment.
